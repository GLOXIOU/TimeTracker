# 🎮 Random Game

**Project still in development**

**Do you want to track your PC screen time?**  
Use TimeTracker, a screen time tracker with a beautiful web interface

**TimeTracker** is a tool that allows you to track total screen time on your PC, for each app, each web tab, and to rewind every year. You can even sort your apps by category !

---

## ✨ Features

* 🔍 Track your screen time effectively
* 🤝 Sort your apps by categories
* ➕ Add categories manually, and customize the app
* 🎲 Do a rewind each year, or at any time
* ⚡ Quick setup, minimal configuration

---

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/GLOXIOU/TimeTracker
   cd TimeTracker
   ```

---

## 🕹️ How It Works
 - You let the app start at every start of your PC
 - The app fetches your screen time in real time
 - It store it in a JSON file
 - You can acces a beautiful web interface
 - You can share your rewind to everyone 🎉
